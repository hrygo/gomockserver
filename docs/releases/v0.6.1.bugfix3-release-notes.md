# v0.6.1.bugfix3 发布说明

## 概述
此版本修复了 Docker 构建和测试环境配置的问题，确保应用程序能够正确构建和运行。

## 修复的问题
1. 修复了 Docker 构建过程中无法找到源代码的问题
2. 修复了测试环境配置中 MongoDB 连接问题
3. 优化了 Dockerfile 中的调试信息输出
4. 改进了测试脚本的稳定性和可靠性

## 技术细节
- 修改了 Dockerfile 和 Dockerfile.test，增强了调试信息输出
- 更新了 config.test.yaml 配置文件，使用 host.docker.internal 连接到主机上的 MongoDB
- 修复了 Dockerfile.test 中配置文件复制的问题
- 创建了完整的测试脚本来验证修复

## 验证
- Docker 构建成功通过
- 应用程序能够正确连接到 MongoDB
- 健康检查端点正常工作
- 测试脚本成功运行

## 文件变更
- `Dockerfile`: 增强调试信息输出
- `Dockerfile.fullstack`: 增强调试信息输出
- `Dockerfile.test`: 修复配置文件复制问题
- `config.test.yaml`: 更新 MongoDB 连接配置
- `test_fix.sh`: 新增测试脚本
